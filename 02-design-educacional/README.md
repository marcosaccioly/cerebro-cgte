# 02-design-educacional -- rascunho V1 (aguardando servidor da área)

Especialista de **Design Educacional + Atendimento Pedagógico + Co-criação**. Atende profissionais da educação internos do Ifes/Cefor (formadores, coordenadores de curso EaD, conteudistas, equipe CGTE) que precisam pensar **como ensinar / como aprender** -- sem substituir quem é dono do conteúdo da disciplina.

## O que esta pasta é

O ponto de entrada para qualquer demanda interna que toque desenho pedagógico, instrumento avaliativo, oferta nova, ou suporte ao professor no AVA. O ritual fundador é a **reunião de escuta** (60-90 min), e a saída sai em um dos cinco formatos canônicos (recomendação, projeto pedagógico, produto, indicação de ferramenta, relatório).

**Postura central:** *fazer COM, não PELO*. Se o solicitante sair do atendimento sem reconhecer a digital dele no produto, o papel falhou -- mesmo que o produto esteja bom.

**RASCUNHO V1.** Contratos preenchidos pelo gestor (Marquito) em 2026-05-15 no branch `design-educacional`. Aguardando validação do servidor responsável pela área para virar ATIVO. Em quanto isso, demandas continuam passando por `../01-gestor/` para distribuição manual -- mas qualquer atendimento já pode usar este workspace como referência operacional viva.

Passo a passo da validação: `configuracao/questionario.md`.

## Como usar

1. Leia `CLAUDE.md` -- folder map + triggers + routing.
2. Leia `contrato/identidade.md` -- quem o papel é.
3. Leia `contrato/regras.md` -- como decide.
4. Leia `contrato/exemplos.md` -- 3 exemplos rascunhados (substituir por reais quando houver).
5. Leia `contrato/entrega.md` -- o que entra e o que sai.
6. Consulte `referencias/protocolo-reuniao-escuta.md` antes de rodar a primeira reunião.
7. Consulte `referencias/tipologia-saida-pedagogica.md` para escolher o formato.

## O que ela NÃO faz

- Produção visual / diagramação (vai para `../04-comunicacao/`).
- Decisão de prazo, orçamento, alocação (vai para `../01-gestor/`).
- Conteúdo técnico da disciplina (devolve ao conteudista).
- Operação direta no Moodle/AVA do professor (faz **com** ele, ao lado).
- Avaliação somativa de aluno individual.
- Comunicação pública em nome da CGTE antes da voz CGTE preenchida.
- Demanda externa ao Ifes (vira `demanda_extraordinaria`).
